# mysql_monitor
MySQL状态监控图形展示

通过简单的配置，可在页面监控到MySQL连接数，QPS，主从状态等信息（类似Lepus）。

# check_mysql_status.php（用来采集被监控端MySQL状态信息）
# check_mysql_repl.php（用来采集被监控端MySQL主从复制信息）
